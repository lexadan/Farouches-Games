# Farouches Games

-   Game Collection the farouche discord

## Getting started

```shell
git clone git@github.com:lexadan/Farouches-Games.git
# init local env file for private tokens
make init-env
```

For developpement
```shell
make build-dev
```


For production
```shell
make build-prod
```
